# vault-addon

Open-source link aggregator. Single Go binary.

## Install

Download from [releases](https://github.com/SlasshyOverhere/vault-addon/releases) or build:

```bash
go build -o vault-addon .
```

## Usage

```bash
./vault-addon --yes
```

Runs on port `51546` by default. Set `PORT` to change.
